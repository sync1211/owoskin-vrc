# Description

This folder contains additional contact prefabs for VRChat's various interaction tags. (e.g. Combat or Elements)
Additionally, this folder contains a set of item examples which make use of these tags.

# Installing the additional contacts

## Installation instructions
1. Drag and drop the prefab(s) onto your avatar and align the colliders with your armature (**Do not** move items out of the prefab)
2. Import the `_Presets.json` file for the sensations you want to add to OWOVRC
3. (Optional, only required for Gesture Manager) Add the matching parameter package to the avatar or import it into an existing parameter set.

## Sensation names

### Combat
* Damage -> Damage
* Damage Blunt -> DamageBlunt
* Damage Sharp -> DamageSharp
* Ammunition -> Shot
* Projectile -> Projectile


### Elements
* Water -> Water
* Wind -> Wind
* Fire -> Fire

### Interaction
* Brush -> Brush
